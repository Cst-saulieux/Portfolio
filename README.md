# Portfolio - 1ère Année BUT Réseaux & Télécommunications

🔗 [Visiter mon Portfolio en ligne](https://cst-saulieux.github.io/Portfolio/)

Bienvenue sur le dépôt de mon portfolio étudiant. Ce projet a été réalisé pour documenter et présenter mon parcours, mes compétences et mes réalisations au cours de ma première année de BUT Réseaux et Télécommunications (R&T) à l'IUT Grand Ouest Normandie.

# 🎯 À propos du projet

Ce site web est un portfolio interactif de type "One-Page". Il a été conçu et développé entièrement de zéro, sans utiliser de bibliothèques externes ou de frameworks (comme Bootstrap ou Tailwind), afin de démontrer une maîtrise solide des fondamentaux du Web.

Il permet de découvrir :

- **Mon profil** : Ma présentation personnelle, mes passions (Skateboard freestyle, sports extrêmes, Warhammer) et mon évolution.

- **Mes projets** (SAÉ) : Une analyse détaillée de mes Mises en Situation d'Apprentissage et d'Évaluation (ex: Hygiène informatique, maquettage de réseau, solutions PME).

- **Mes bilans** de compétences : Une auto-évaluation de ma progression sur les compétences fondamentales (Administrer les réseaux, Connecter les entreprises, Créer des outils informatiques).

Mon projet professionnel : Mon choix de réorientation assumé et réfléchi vers un BTS CIEL.

# 🚀 Évolution et Projet Professionnel

Bien que cette première année en BUT R&T ait été extrêmement enrichissante et m'ait permis d'acquérir de fortes bases techniques, elle m'a également aidé à affiner mes aspirations. M'étant rendu compte que le domaine spécifique des télécommunications ne correspondait pas à mes objectifs de carrière, j'ai décidé de me réorienter vers un BTS CIEL (Cybersécurité, Informatique et réseaux, Électronique).

Ce choix représente pour moi la meilleure option pour me concentrer exclusivement sur ce qui me passionne : l'informatique pure, la cybersécurité et l'administration réseau.

# 🛠️ Technologies Utilisées

**HTML5** : Structure sémantique de la page.

**CSS3** : Mise en page responsive (Flexbox, CSS Grid), utilisation de variables natives (:root), animations fluides et design adaptatif (Media Queries) pour mobile et desktop.

**JavaScript (Vanilla)** : Manipulation du DOM pour gérer la navigation dynamique entre les différentes sections du site ("Single Page Application" logic) sans rechargement de page.

# 📂 Structure des fichiers

Le projet est volontairement minimaliste et optimisé :

**index.html** : Contient l'intégralité du code (Structure HTML, Styles CSS et Logique JavaScript).

**imgmoi.jpg** : Photo de profil utilisée dans l'interface.

**README.md** : Ce fichier de documentation.

# 💻 Installation et exécution locale

Pour consulter ce projet en local sur votre machine :

Clonez ce dépôt ou téléchargez les fichiers.

Assurez-vous que le fichier imgmoi.jpg se trouve dans le même répertoire que index.html.

Ouvrez simplement le fichier index.html avec n'importe quel navigateur Web moderne (Chrome, Firefox, Safari, Edge). Aucune installation de serveur n'est requise.
